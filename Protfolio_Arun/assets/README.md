# Arun Interiors Portfolio Assets Directory

Place all your image assets, photos, logos, and icons in this folder to load them easily in the app.

## Folders Created:
- `/images/` - Place general background or about images here.
- `/projects/` - Place your 3D design diaries and portfolio project photos here.
- `/logos/` - Place your brand logos and icons here.

## How to import in Code:
```javascript
import projectImage from '../../assets/projects/project1.jpg';
```
